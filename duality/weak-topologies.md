# Weak Topologies

:::{tip} Big Idea
The Hahn-Banach theorem gives us a rich supply of continuous linear functionals.
Each functional defines a family of parallel hyperplanes (a foliation). The
**weak topology** is what you get when you use these foliations as your notion of
"nearness": two points are close if every functional gives them similar readings.
The resulting open sets are **slabs** between parallel hyperplanes, unbounded in
most directions. Fewer open sets means it is easier for sets to be closed, and
easier for sets to be compact. This is the geometric reason that the unit ball,
which is never norm-compact in infinite dimensions, can become compact in the
weak or weak\* topology.
:::

## From norm balls to weak neighborhoods

In a normed space $X$, the **norm topology** is generated by the open balls

$$B_r(x) = \{ y \in X : \|y - x\| < r \}.$$

These are bounded in every direction. In finite dimensions, closed bounded sets
are compact (Heine-Borel). In infinite dimensions, the closed unit ball is
**never** norm-compact (Riesz's theorem), because there is room for sequences to
spread out in infinitely many orthogonal directions without clustering.

The idea behind the weak topology is to relax the notion of neighborhood. The
building blocks are **slabs**: for a single functional $f \in X^*$ and
$\varepsilon > 0$, define

$$S(f, x, \varepsilon) = \{ y \in X : |f(y - x)| < \varepsilon \}.$$

Geometrically, this is the region between the two parallel hyperplanes $f(y) =
f(x) - \varepsilon$ and $f(y) = f(x) + \varepsilon$: a "thickened hyperplane"
centered at the level set through $x$. It constrains $y$ only in the direction
that $f$ measures, and is **unbounded** in every direction within $\ker f$.

A weak open set is built up in three layers:

1. **Sub-basic open sets (slabs).** A single slab $S(f, x, \varepsilon)$
   imposes one linear constraint. In $\ell^2$, the slab $\{y : |y_1| < 1\}$
   constrains the first coordinate but places no restriction on $y_2, y_3,
   \ldots$ — it is an infinite strip extending in all other directions.

2. **Basic open sets (finite intersections of slabs).** A finite intersection

   $$U(x; f_1, \ldots, f_n, \varepsilon) = \bigcap_{i=1}^n S(f_i, x, \varepsilon)$$

   constrains $n$ directions simultaneously. In $\ell^2$, the set $\{y : |y_1|
   < 1 \text{ and } |y_2| < 1\}$ constrains two coordinates but is still
   unbounded in the $e_3, e_4, \ldots$ directions. Think of this as a **tube**:
   bounded cross-section in finitely many directions, infinite extent in the
   rest. These form a **neighborhood basis** for the weak topology.

3. **General open sets (arbitrary unions of basic open sets).** Every weakly
   open set is a union of tubes. The shapes can be more complicated, but the
   tubes are the fundamental building blocks.

The key point: in infinite dimensions, every basic neighborhood is unbounded
(since $n < \dim X$ always), so no bounded set like a norm ball can contain a
weak neighborhood. This means norm-open balls are **not** weakly open.

```{prf:definition} Weak topology
:label: weak-topology-def

The **weak topology** on $X$, denoted $\sigma(X, X^*)$, is the coarsest topology
making every $f \in X^*$ continuous. A neighborhood basis at $x \in X$ consists
of the basic open sets:

$$U(x; f_1, \ldots, f_n, \varepsilon) = \{ y \in X : |f_i(y - x)| < \varepsilon \text{ for } i = 1, \ldots, n \}$$

where $f_1, \ldots, f_n \in X^*$ and $\varepsilon > 0$.
```


## Example: the weak topology on $\ell^2$

Take $X = \ell^2$ with its standard orthonormal basis $(e_n)$. By the Riesz
representation theorem, every $f \in (\ell^2)^*$ is of the form $f(x) =
\langle x, a \rangle$ for some $a \in \ell^2$. Consider the functional

$$f_1(x) = \langle x, e_1 \rangle = x_1.$$

The slab $S(f_1, 0, 1) = \{x \in \ell^2 : |x_1| < 1\}$ is the region between
the hyperplanes $x_1 = -1$ and $x_1 = 1$. This slab contains vectors with
arbitrarily large norm, as long as their first component is small.

```{prf:example} Inside the weak neighborhood but outside the norm ball
:label: weak-slab-example
:class: dropdown

The vector $y_n = \tfrac{1}{2} e_1 + n \, e_2$ satisfies $|f_1(y_n)| =
\tfrac{1}{2} < 1$, so $y_n \in S(f_1, 0, 1)$ for every $n$. But $\|y_n\| =
\sqrt{1/4 + n^2} \to \infty$. The weak slab sees only the first coordinate and
is blind to what happens in the $e_2$ direction.
```

Adding more functionals narrows the neighborhood but never makes it bounded.
The neighborhood $U(0; f_1, \ldots, f_n, \varepsilon)$ constrains $n$
coordinates but leaves infinitely many unconstrained.

## Weak neighborhoods as slabs between hyperplanes

The connection to the hyperplane geometry from earlier sections is direct. A
functional $f \in X^*$ defines a family of parallel hyperplanes

$$H_c = \{ x \in X : f(x) = c \}, \quad c \in \mathbb{R}.$$

The slab $S(f, x_0, \varepsilon)$ is the region between $H_{f(x_0) -
\varepsilon}$ and $H_{f(x_0) + \varepsilon}$: a strip of width
$2\varepsilon / \|f\|$ in the direction normal to the hyperplanes, extending
infinitely in all directions parallel to them.

A basic weak neighborhood $U(x_0; f_1, \ldots, f_n, \varepsilon)$ is the
intersection of $n$ such slabs. Each slab constrains one "measurement
direction." The neighborhood is bounded only in the subspace spanned by the
normals to these $n$ hyperplanes, and unbounded in the (infinite-dimensional)
complement.

```{prf:remark} Why fewer open sets helps
:label: fewer-open-sets-remark
:class: dropdown

A topology with fewer open sets makes it:

- **Easier to be closed.** Fewer open sets means fewer unions of open sets to
  avoid, so more sets qualify as closed. In particular, the closed unit ball
  stays closed in the weak topology (it is closed in the norm topology and the
  weak topology is coarser).

- **Easier to be compact.** Compactness requires that every open cover has a
  finite subcover. With fewer open sets, there are fewer covers to worry about,
  so it is easier for a set to be compact.

- **Harder to be open.** Norm-open balls are generally *not* weakly open in
  infinite dimensions (they would need to contain an unbounded weak
  neighborhood, which they cannot).

This is the tradeoff: the weak topology sacrifices the ability to separate
points by open sets in the norm sense, but gains compactness of bounded sets.
```

```{prf:remark} The unit ball as an intersection of slabs
:label: unit-ball-intersection-remark
:class: dropdown

The slab picture gives a clean characterization of the unit ball. Each $f \in
X^*$ with $\|f\| \leq 1$ defines a closed slab $\{x : |f(x)| \leq 1\}$, and
the unit ball fits inside every such slab. The
{prf:ref}`sup formula <hb-sup-formula>` gives the converse:

$$B_X = \bigcap_{\|f\| \leq 1} \{x : |f(x)| \leq 1\}.$$

The unit ball is exactly the intersection of all dual slabs. This means $B_X$ is
**weakly closed** (it is an intersection of weakly closed sets, since each $f$
is weakly continuous). It also explains the connection to Banach-Alaoglu: the
same slab structure that defines the weak topology is what makes the unit ball
compact in it.
```


## The weak\* topology

The weak\* topology on $X^*$ is defined by the same slab construction, but the
functionals doing the slicing come only from $X$ (via the canonical embedding),
not from all of $X^{**}$.

```{prf:definition} Weak* topology
:label: weak-star-topology-def

The **weak\* topology** on $X^*$, denoted $\sigma(X^*, X)$, is the coarsest
topology making every evaluation map $\hat{x} : X^* \to \mathbb{R}$, $\hat{x}(f)
= f(x)$, continuous. A basis of open neighborhoods of $f \in X^*$ is:

$$V(f; x_1, \ldots, x_n, \varepsilon) = \{ g \in X^* : |g(x_i) - f(x_i)| < \varepsilon \text{ for } i = 1, \ldots, n \}$$

where $x_1, \ldots, x_n \in X$ and $\varepsilon > 0$.
```

The slabs are now in $X^*$: the hyperplane $\{g \in X^* : g(x) = c\}$ is the
set of all functionals giving reading $c$ on the object $x$. A weak\* slab
constrains the values of $g$ on finitely many test objects.


## Example: the weak\* topology on $\ell^1 = (c_0)^*$

Take $X = c_0$ (sequences converging to $0$ with sup norm). Its dual is $X^* =
\ell^1$ with $f(x) = \sum_k f_k x_k$ for $f = (f_k) \in \ell^1$, $x = (x_k)
\in c_0$.

The **norm ball** in $\ell^1$ is $B_1 = \{f \in \ell^1 : \sum_k |f_k| \leq 1\}$,
the familiar cross-polytope (diamond) in finite-dimensional slices.

Now take the test object $x = e_1 = (1, 0, 0, \ldots) \in c_0$. The evaluation
$\hat{e}_1(f) = f_1$ defines a weak\* slab

$$V_1 = \{ f \in \ell^1 : |f_1| < 1 \}.$$

This slab constrains only the first component of $f$. A sequence like $f^{(n)}
= \tfrac{1}{2} e_1 + e_n$ has $|f^{(n)}_1| = \tfrac{1}{2} < 1$ (inside the
slab) but $\|f^{(n)}\|_1 = \tfrac{3}{2}$ (outside the norm ball).

```{prf:example} Weak\* neighborhoods in $\ell^1$
:label: weak-star-slab-ell1
:class: dropdown

The weak\* neighborhood $V(0; e_1, \ldots, e_n, \varepsilon) = \{f \in \ell^1 :
|f_k| < \varepsilon \text{ for } k = 1, \ldots, n\}$ constrains the first $n$
components but places no restriction on $f_k$ for $k > n$. In infinite
dimensions, every weak\* neighborhood of $0$ is unbounded in $\ell^1$.
```


## The hierarchy of topologies

Since $J(X) \subseteq X^{**}$, every weak\* open set is determined by evaluation
against elements of $X$, while weak open sets in $X^*$ are determined by
evaluation against all of $X^{**}$. More evaluation functionals means more slabs,
which means more open sets:

$$\text{weak* topology} \subseteq \text{weak topology on } X^* \subseteq \text{norm topology on } X^*.$$

Each inclusion means "coarser than or equal to." The first inclusion is strict
when $X$ is not reflexive (there exist elements of $X^{**} \setminus J(X)$ that
generate extra slabs). When $X$ is reflexive, $J(X) = X^{**}$ and the weak\*
and weak topologies on $X^*$ coincide.

```{prf:remark} The compactness payoff
:label: compactness-payoff-remark
:class: dropdown

The coarser the topology, the easier it is for sets to be compact. This is why
compactness improves as we move left in the chain:

- **Norm topology**: the closed unit ball of $X^*$ is compact only if $X^*$ is
  finite-dimensional.
- **Weak topology on $X^*$**: the closed unit ball is compact if and only if
  $X^*$ is reflexive.
- **Weak\* topology on $X^*$**: the closed unit ball is **always** compact
  (Banach-Alaoglu).

The weak\* topology is the coarsest of the three, so it is the easiest setting
for compactness. This is exactly why Banach-Alaoglu works without any
reflexivity assumption.
```


## The canonical embedding revisited

The relationship between the three topologies is mediated by the canonical
embedding $J : X \to X^{**}$. Each $x \in X$ defines an evaluation functional
$J[x] \in X^{**}$ by $J[x](f) = f(x)$.

- The **weak topology on $X$** is $\sigma(X, X^*)$: slabs come from $f \in X^*$.
- The **weak\* topology on $X^*$** is $\sigma(X^*, X)$: slabs come from
  $J[x]$ for $x \in X$.
- The **weak topology on $X^*$** is $\sigma(X^*, X^{**})$: slabs come from all
  $\phi \in X^{**}$, including those not in $J(X)$.

When $X$ is reflexive, $J$ is surjective, so the weak\* and weak topologies on
$X^*$ are generated by the same set of slabs and therefore coincide.

```{prf:example} $c_0 \hookrightarrow \ell^1 \hookrightarrow \ell^\infty$
:label: c0-embedding-example
:class: dropdown

Take $X = c_0$. Then $X^* = \ell^1$ and $X^{**} = \ell^\infty$. The canonical
embedding $J : c_0 \hookrightarrow \ell^\infty$ sends $x = (x_k) \in c_0$ to
the same sequence viewed in $\ell^\infty$. Since $c_0 \subsetneq \ell^\infty$
(e.g., the constant sequence $(1, 1, 1, \ldots) \in \ell^\infty \setminus c_0$),
the space $c_0$ is not reflexive.

The weak\* topology on $\ell^1$ uses slabs from $c_0$: neighborhoods constrain
$f(x)$ for $x \in c_0$. The weak topology on $\ell^1$ uses slabs from
$\ell^\infty$: neighborhoods can additionally constrain $f$ against sequences
that do not converge to $0$. The weak topology is strictly finer, with more
open sets and therefore fewer compact sets.
```
